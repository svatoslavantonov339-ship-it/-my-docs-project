# Спецификация эндпоинтов API

## 1. Получение аналитики абонемента студента
* **URL:** `/api/v1/students/{student_id}/subscription`
* **Метод:** `GET`

### Параметры запроса (Path):
| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| `student_id` | `integer` | Номер студенческого билета (например, `202601`) |
