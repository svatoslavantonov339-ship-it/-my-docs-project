# Быстрый старт для разработчика

В данном разделе описан процесс локального запуска бэкенда `Library Core` для проведения тестирования.

## Системные требования
* Python версии 3.10 или выше.
* Установленный менеджер пакетов `pip`.

## Порядок локального развертывания

1. Установите необходимые зависимости для работы асинхронного веб-сервера:
1. Установите зависимости: `pip install fastapi uvicorn pydantic`
2. Запустите сервер: `uvicorn main:app --reload`
3. Интерактивная документация Swagger доступна по адресу: http://127.0.0.1:8000/docs
