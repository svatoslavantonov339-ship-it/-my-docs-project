# Система Digital Library Core API

Добро пожаловать в техническую документацию микросервиса управления цифровым фондом учебных материалов. Данный сервис обеспечивает бесшовную интеграцию между распределенным хранилищем книг и внутренними системами университета.

## Основные возможности системы:
* **Автоматический учет:** Контроль выдачи и возврата учебных пособий.
* **Аналитика абонементов:** Оценка загруженности студентов и расчет времени на освоение литературы.
* **Высокая производительность:** Асинхронный опрос узлов хранения без блокировки основного потока.

Для быстрого развертывания локальной копии перейдите в раздел [Быстрый старт](dev/quickstart.md).
